# Qt Projects 📦

使用 C++ 和 Qt 开发的两个学习项目：一个简单的计算器和一个照片显示器。旨在熟悉 Qt 框架、CMake 构建流程以及跨平台 GUI 应用开发。


## 项目一：Calculator_Qt 🧮

一个基本的图形界面计算器，支持加减乘除运算。

### 功能特性
- 支持整数和小数运算
- 实时显示计算结果
- 清除与退格功能

### 技术栈
- Qt Widgets
- C++
- CMake

### 快速开始
```bash
cd calculator_qt
mkdir build && cd build
cmake ..
make
./CalculatorApp
```
